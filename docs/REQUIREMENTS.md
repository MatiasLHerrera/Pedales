# Requisitos del proyecto

## Convenciones de estado

- APROBADO
- PROPUESTO
- PENDIENTE
- DESCARTADO
- REQUIERE VALIDACIÓN

## Requisitos aprobados inicialmente

- ESP32-S3-WROOM-1-N16R8;
- CS4272;
- USB-C nativo;
- I2S para audio;
- I2C para control;
- 48 kHz;
- 24 bits efectivos;
- slots de 32 bits previstos;
- ESP32 como maestro inicial;
- CS4272 como esclavo inicial;
- VA a 5 V;
- VD y VL a 3,3 V;
- PCB inicial de dos capas;
- futura revisión posible de cuatro capas;
- placas analógicas separadas;
- montaje manual;
- preferencia por 0805.

## Requisitos pendientes

- regulador de 3,3 V;
- protección USB;
- conectores entre placas;
- presupuesto de alimentación para placas externas;
- asignación definitiva de GPIO;
- arquitectura de reloj definitiva;
- dimensiones de la PCB;
- fabricante;
- selección de componentes analógicos;
- arquitectura de firmware.
