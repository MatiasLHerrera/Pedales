# Pedales

Proyecto modular de hardware y firmware para procesamiento de audio de guitarra eléctrica.

## Objetivos generales

- desarrollar diferentes placas electrónicas;
- construir una mainboard basada inicialmente en ESP32-S3 y CS4272;
- experimentar con preamplificadores y etapas analógicas;
- implementar efectos digitales;
- comparar distintas arquitecturas;
- documentar y validar cada etapa.

## Estado actual

Fase de documentación y definición de requisitos.

Existe una jerarquía KiCad `structure-only` para MB-001, pero todavía no existe
diseño eléctrico aprobado, PCB ruteado ni firmware.

## Método de trabajo

El proyecto se desarrolla mediante tareas pequeñas, revisión intermedia y documentación persistente para ChatGPT y Codex.

## Estructura actual

- `docs`: alcance, requisitos, arquitectura, decisiones y fuentes del proyecto.
- `references`: documentación oficial, diseños de referencia y proyectos de inspiración, organizados por procedencia.

La estructura crecerá progresivamente a medida que las tareas del proyecto lo requieran.
