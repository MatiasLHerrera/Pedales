# Área local de documentación técnica

Esta carpeta es un área local de importación y auditoría para copias de documentos técnicos utilizados por el proyecto Pedales.

Los archivos de fabricantes deben tratarse como documentos de solo lectura: no se deben editar sus contenidos ni sus metadatos. Los binarios técnicos, incluidos PDF, archivos comprimidos, modelos STEP y dibujos DXF, no deben subirse automáticamente a GitHub.

El catálogo documental oficial del proyecto es [`docs/SOURCES.md`](../docs/SOURCES.md). Allí se registran procedencia, versión, fecha, integridad, hash SHA-256, estado de verificación y uso previsto de cada fuente.

La ubicación definitiva de los documentos locales se decidirá en una tarea posterior. Hasta entonces, no se deben mover, renombrar, sustituir ni borrar los originales auditados. Tampoco deben almacenarse duplicados sin una justificación documentada en el catálogo.
